# ⚡ Electrician Hub

Electrician Hub is an educational web platform designed to help students and professionals learn electrical concepts in one place. The website provides structured study materials, interactive quizzes, component information, and downloadable PDF notes.

## ✨ Features

* 📚 Electrician Chapters
* 🔌 Electrical Components
* 📝 Interactive Quizzes
* 📄 Downloadable PDF Notes
* 📖 Easy-to-understand Learning Content
* 📱 Responsive Design for Mobile and Desktop
* 🚀 Fast and User-Friendly Interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript


## 🎯 Purpose

The goal of this project is to make electrician learning simple, organized, and accessible. It serves as a free learning resource for students preparing for electrician exams and anyone interested in electrical engineering fundamentals.
