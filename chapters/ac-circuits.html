Chapter 1: AC Aur DC Fundamentals

What is AC (Alternating Current)? [Interview Framework]
Alternating Current (AC) is an electric current that changes both its magnitude and direction periodically over time.
Hinglish Definition: AC wo current hota hai jo samay (time) ke saath apni direction aur value (parimaan) dono ko lagatar periodically change karta rehta hai.
💡 AC Real-Life Examples:
• Household Grid Power Supply = 230V, 50Hz AC.
• Large scale Alternators aur Power Plants Generators AC hi produce karte hain.
What is DC (Direct Current)?
DC (Direct Current) is the electric current which flows consistently in only one unidirectional path.
Examples: Chemical Battery Cell, Solar Photovoltaic Cell, aur aapke Smartphones ki Lithium-Ion battery hamesha pure steady DC supply hi deti hai.

AC Waveform Metric Terms

1. Cycle (चक्र):

Ek complete positive half-wave aur ek complete negative half-wave milkar jo poora single periodic path banati hai, use **1 Cycle** kehte hain.

2. Frequency (f) Aur Time Period (T):
Frequency is defined as the total number of cycles completed by the wave in one second. Unit: Hertz (Hz).
One complete cycle ko poora karne me network ko jitna time lagta hai, use **Time Period (T)** bolte hain. Dono inversely proportional hote hain:
f = 1 / T | T = 1 / f
3. Angular Frequency (ω):

Iski angular measurement speed unit **rad/s** hoti hai:

ω = 2πf

Peak, RMS Aur Average Constants

Peak Value (Maximum Value):

AC wave ka zero line se upar ya niche jo maximum amplitude/crest point hota hai use Peak Value kehte hain (`Vm` ya `Im`).

RMS Value (Root Mean Square) [Most Critical Factor]:
RMS value is the effective value of AC that produces the exact same heat output as a steady DC supply under equal resistance.
Vrms = Vm / √2 = 0.707 × Vm
Irms = Im / √2 = 0.707 × Im
Average Value (औसत मान):
Vavg = 0.637 × Vm
Form Factor & Peak Factor Multipliers:

• **Form Factor:** RMS Value / Average Value = **1.11** (Standard Sine Wave)
• **Peak Factor:** Maximum Value / RMS Value = **1.414**

🎯 Formula Shortcut Cheat-sheet:
• **RMS** = 0.707 × Max Value
• **Average** = 0.637 × Max Value
• **Form Factor** = 1.11 Constant
• **Peak Factor** = 1.414 Constant
• **Sinusoidal Equation:** v = Vm sin(ωt)

AC Behavior Across Pure Circuit Elements (R, L, C)

Pure Element Load Phase Configuration Angle Power Factor Effect Reactance Equation Matrix
Pure Resistance (R) Voltage aur Current **Same Phase (0°)** me linear hote hain. Exactly **Unity (1.0)** R (Ohm Opposition)
Active Power: P = V × I
Pure Inductor (L) Current Current Voltage se **90° Lag** (piche) rehta hai. **Zero Lagging** Inductive Reactance: XL = 2πfL (Ohm)
Pure Capacitor (C) Current Current Voltage se **90° Lead** (aage) nikal jata hai. **Zero Leading** Capacitive Reactance: XC = 1 / (2πfC) (Ohm)
🚀 Quick Exam Keys:
• **L = Lag:** Inductor core winding hamesha current ko lag karwayegi.
• **C = Current Leads:** Capacitor layers me current always phase me lead karega.

Impedance Vector Network & Power Triangle

Impedance (Z) Analytical Concept:

AC line circuit me Ohm's Resistance (R) aur Reactor Components (XL/XC) ke vector total opposition ko **Impedance** bolte hain. Unit: **Ohm**.

Z = √(R² + (XL - XC)²) | AC Ohm's Law: V = I × Z
AC Power Triangle System Components:

1. **Active Power (P):** Actual dynamic work generator. Formula: `P = VI cosϕ` (Unit: Watt / kW)
2. **Reactive Power (Q):** Magnetic flux field creator. Formula: `Q = VI sinϕ` (Unit: VAR / kVAR)
3. **Apparent Power (S):** Combined gross vector input. Formula: `S = VI` (Unit: VA / kVA)

Vector Power Matrix: S² = P² + Q²

Three-Phase System Balancing: Star Aur Delta Connections

Bade industrial dynamic electrical loads ko handle karne ke liye hum Poly-Phase Systems standard framework use karte hain, jahan teeno active hot parameters ke beech uniform 120° phase space difference rehta hai. Line voltage matrix standard value **415V** hai.

⭐ Star Connection Architecture (Y-Layout):

Isme single shared common point hota hai jise **Neutral Node** kehte hain. Phase system distribution line balance ke liye best hai.

VL = √3 × Vph | IL = Iph
🔺 Delta Connection Architecture (Δ-Layout):

Isme koi separate unique Neutral path configuration nahi hoti. Industrial induction heavy phase line systems me deployment hoti hai.

VL = Vph | IL = √3 × Iph
Combined Balanced 3-Phase Power Vector Formula:
P = √3 × VL × IL × cosϕ
